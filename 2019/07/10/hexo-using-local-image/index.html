HEXO 使用本地图片

刚开始使用HEXO的时候，为了简单使用，将图片直接存放到图床上面，之前使用的是 https://sm.ms/图床工具，
但是后来考虑到如果图床保留的图片都失效了，这会导致图片全部不能正常显示，所以就考虑转换到使用本地图片，这样不会导致图片失效
这里使用hexo-asset-image 插件
